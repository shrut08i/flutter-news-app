# Flutter News App

## Description

A Flutter news application that fetches top headlines using NewsAPI and supports offline caching using Hive.

## Features

* Fetch latest news headlines
* Search news articles
* Offline caching
* Clean UI

## Requirements

* Flutter SDK installed
* Internet connection (for first fetch)

## How to Run

1. Clone or extract the project.
2. Navigate to the project folder.

```bash
flutter pub get
flutter run
```
